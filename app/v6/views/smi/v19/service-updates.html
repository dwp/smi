Service update 31 July 2018

 

Warning You need to set up a new shortcut for Support for Mortgage Interest.

Create a shortcut

  1. Right click on your desktop.
  2. Select ‘New Shortcut’.
  3. Where it says ‘Type the location of the item:’ copy and paste this web address (URL): http://smi-admin.gov.uk/
  4. Select ‘Next’.
  5. Where it says ‘Type the name for this shortcut’, enter Manage a mortgage interest loan.
  6. Select ‘Finish’.
  7. The shortcut will appear on your desktop. Double click on it to access the Manage a mortgage interest loan system.

You’ve been automatically redirected for now. But on 30 October 2018 the old site will be deleted and any old shortcuts will no longer work.
